Se roba grossa: (user story)
  prima:
    git branch nome-branch
    git checkout nome-branch
  dopo:
    git add file1 file2 … filen
    git commit -m "descrizione-commit"
    git checkout develop
    git merge nome-branch
    git push

Se roba piccola: (es. yaml)
  prima:
    git checkout develop
  dopo:
    git add file1 file2 … filen
    git commit -m "descrizione-commit"
    git checkout master
    git merge develop


Quando tutto è funzionante:
  post lavoro:
    git checkout master
    git merge develop
